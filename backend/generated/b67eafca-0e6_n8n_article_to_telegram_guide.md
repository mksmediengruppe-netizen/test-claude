# N8N Workflow: Article Rewriter & Telegram Publisher

## 📋 Описание
Готовый workflow для автоматического сбора статей с сайтов, переписывания через OpenAI GPT-4 и публикации в Telegram группе.

## 📁 Файлы
1. **Workflow JSON**: `article_to_telegram_workflow.json` - основной файл для импорта в n8n
2. **Инструкция**: Этот документ

## 🚀 Установка и настройка

### 1. Импорт workflow в n8n
1. Откройте n8n интерфейс
2. Перейдите в "Workflows" → "Import from File"
3. Загрузите файл `article_to_telegram_workflow.json`

### 2. Настройка Credentials

#### OpenAI API:
1. В n8n перейдите в "Credentials" → "New"
2. Выберите "OpenAI API"
3. Введите:
   - **Name**: `OpenAI API`
   - **API Key**: Ваш ключ от OpenAI
4. Сохраните

#### Telegram Bot:
1. Создайте бота через @BotFather в Telegram
2. Получите токен бота
3. В n8n создайте новые credentials:
   - **Type**: Telegram API
   - **Name**: `Telegram Bot Token`
   - **Access Token**: Токен вашего бота
4. Добавьте бота в группу и сделайте администратором

### 3. Настройка параметров

#### Telegram группа:
1. В узле "Telegram Send" замените `-1001234567890` на ID вашей группы
   - Чтобы получить ID группы: добавьте @RawDataBot в группу и отправьте `/id`

#### Источники статей:
Workflow поддерживает 3 режима:

1. **Ручной запуск** (Manual Trigger):
   - Укажите URL статьи в узле "Manual Trigger"
   - Запустите вручную

2. **Автоматический по расписанию** (Schedule Trigger):
   - Настроено на публикацию в 10:00, 16:00, 20:00
   - Берет случайную статью с Hacker News

3. **Кастомный источник**:
   - Измените URL в узле "HTTP Request - Get Article"

## 🔧 Конфигурация узлов

### 1. HTTP Request - Get Article
- Получает HTML страницы
- Можно изменить URL на любой сайт

### 2. HTML Parser
- Извлекает заголовок и контент
- Удаляет рекламу, скрипты, комментарии
- Поддерживает популярные селекторы статей

### 3. OpenAI Rewriter
- Переписывает статью через GPT-4
- Делает контент уникальным и оптимизированным для Telegram
- **Важно**: Настройте credentials OpenAI

### 4. Content Formatter
- Форматирует текст для Telegram
- Добавляет Markdown разметку
- Обрезает до 4000 символов (лимит Telegram)

### 5. Telegram Send
- Публикует в группу
- Использует Markdown форматирование
- **Важно**: Настройте chat_id и credentials

## ⚙️ Дополнительные настройки

### Изменение расписания:
1. Откройте узел "Schedule Trigger"
2. Измените время публикации
3. Можно добавить больше временных слотов

### Добавление новых источников:
1. Создайте новый HTTP Request узел
2. Настройте парсинг HTML
3. Подключите к основному workflow

### Настройка промпта OpenAI:
В узле "OpenAI Rewriter" можно изменить system prompt:
```javascript
{
  role: 'system',
  content: 'Ваш кастомный промпт здесь'
}
```

## 🚨 Обработка ошибок

Workflow включает обработку ошибок:
1. **Ошибка OpenAI** → Отправка сообщения об ошибке в Telegram
2. **Ошибка парсинга** → Fallback на оригинальный контент
3. **Ошибка Telegram** → Логирование в консоль

## 📊 Мониторинг

### Встроенное логирование:
- Узел "Success Logger" - логирует успешные публикации
- Узел "Error Logger" - логирует ошибки

### Внешний мониторинг:
1. Настройте n8n webhooks для уведомлений
2. Используйте Telegram бота для уведомлений об ошибках
3. Настройте мониторинг через Health Checks

## 🔄 Автоматизация

### Варианты триггеров:

2. **Email Trigger** - публикация по получению email

## 💡 Советы по оптимизации

1. **Кэширование**: Добавьте узел "Memory" для кэширования обработанных URL
2. **Очередь**: Используйте "Queue" узел для обработки множества статей
3. **Рейт-лимит**: Настройте задержки между запросами к сайтам
4. **Ротация User-Agent**: Изменяйте заголовки запросов

## 🆘 Устранение неполадок

### Проблема: OpenAI не отвечает
**Решение:**
- Проверьте API ключ
- Проверьте баланс аккаунта
- Увеличьте timeout в настройках узла

### Проблема: Telegram не отправляет сообщения
**Решение:**
- Проверьте chat_id группы
- Убедитесь, что бот - администратор
- Проверьте токен бота

### Проблема: Не парсится контент
**Решение:**
- Измените селекторы в HTML Parser
- Добавьте delay между запросами
- Проверьте robots.txt сайта
