# Git Automation Scripts

Два bash скрипта для автоматизации git workflow.

---

## 📦 Скрипты

### 1. git-auto.sh (Расширенная версия)

Полнофункциональный скрипт с цветным выводом и гибкими опциями.

**Возможности:**
- ✅ Добавление всех файлов или конкретных паттернов
- ✅ Автоматическое добавление даты в сообщение коммита
- ✅ Push в указанную ветку или текущую
- ✅ Цветной вывод статуса операций
- ✅ Проверка на наличие изменений
- ✅ Отображение списка добавленных файлов
- ✅ Показ хеша коммита

**Использование:**

```bash
# Добавить все файлы, закоммитить и запушить в текущую ветку
./git-auto.sh "Update files"

# Добавить только .js файлы
./git-auto.sh "*.js" "Fix bugs"

# Запушить в конкретную ветку (main)
./git-auto.sh "Update" main

# Добавить конкретные файлы
./git-auto.sh "file1.txt file2.txt" "Add files"
```

---

### 2. git-simple.sh (Простая версия)

Минималистичный скрипт для быстрого коммита.

**Использование:**

```bash
./git-simple.sh "Update files"
```

**Что делает:**
1. `git add .` — добавляет все файлы
2. `git commit -m "[2025-03-13] Update files"` — коммит с датой
3. `git push origin <current-branch>` — пуш в текущую ветку

---

## 📝 Формат сообщения коммита

```
[YYYY-MM-DD] Ваше сообщение
```

Пример:
```
[2025-03-13] Fix login bug
```

---

## 🔧 Установка

Скрипты уже установлены на сервере `2.56.240.170` в `/root/`:

- `/root/git-auto.sh` — расширенная версия
- `/root/git-simple.sh` — простая версия

Оба скрипта имеют права на выполнение.

---

## 📋 Примеры работы

### Пример 1: Быстрый коммит
```bash
cd /path/to/your/repo
/root/git-simple.sh "Add new feature"
```

**Вывод:**
```
=== Git Auto Workflow ===
[1/3] Adding all files...
[2/3] Committing changes...
[3/3] Pushing to main...
✓ Done!
  Branch: main
  Date: 2025-03-13 19:15:30
```

### Пример 2: Коммит конкретных файлов
```bash
/root/git-auto.sh "*.py" "Update Python scripts"
```

### Пример 3: Пуш в develop ветку
```bash
/root/git-auto.sh "Hotfix" develop
```

---

## ⚠️ Требования

- Git должен быть инициализирован в репозитории
- Настроен remote `origin`
- Есть права на push в репозиторий
