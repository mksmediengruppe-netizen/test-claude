## 📋 Пошаговое объяснение

### 1️⃣ Schedule Trigger (Триггер расписания)

**Тип:** `n8n-nodes-base.scheduleTrigger`

**Функция:** Запускает workflow по расписанию

**Настройки:**
- Интервал: Каждые 1 час
- Можно изменить на: каждые 30 минут, 2 часа, 6 часов и т.д.

**Зачем нужен:** Автоматический запуск без ручного вмешательства

---

### 2️⃣ RSS Feed Reader (Чтение RSS ленты)

**Тип:** `n8n-nodes-base.rssFeedRead`

**Функция:** Получает новости из RSS источника

**Настройки:**
- URL: `https://news.ycombinator.com/rss` (пример)
- Можно заменить на любой RSS: TechCrunch, Habr, BBC и т.д.

**Примеры RSS источников:**
- Habr: `https://habr.com/ru/rss/articles/?fl=ru`
- TechCrunch: `https://techcrunch.com/feed/`
- BBC: `http://feeds.bbci.co.uk/news/rss.xml`
- Вести: `https://www.vesti.ru/vesti.rss`

**Выходные данные:**
```json
{
  "title": "Заголовок новости",
  "link": "https://example.com/article",
  "description": "Краткое описание",
  "pubDate": "2024-01-15T10:00:00Z",
  "contentSnippet": "Текст новости..."
}
```

---

### 3️⃣ Filter New Posts (Фильтр новых постов)

**Тип:** `n8n-nodes-base.filter`

**Функция:** Отбирает только свежие новости

**Условие:**
```
pubDate > (текущее время - 2 часа)
```

**Зачем нужен:**
- Избегает дубликатов
- Публикует только актуальные новости
- Можно изменить интервал фильтрации

---

### 4️⃣ Prepare Data for AI (Подготовка данных для AI)

**Тип:** `n8n-nodes-base.code`

**Функция:** Формирует промпт для OpenAI

**Что делает:**
1. Извлекает заголовок, описание, ссылку
2. Создаёт структурированный промпт
3. Добавляет инструкции для AI

**Промпт для AI:**
```
Перепиши эту новость для Telegram канала. Требования:
1. Сделай заголовок более кликабельным и интересным
2. Напиши краткое содержание (2-3 предложения)
3. Добавь эмодзи для визуального оформления
4. Используй форматирование Markdown
5. В конце добавь ссылку на источник

Оригинальная новость:
Заголовок: {title}
Содержание: {description}

Ссылка: {link}
```

---

### 5️⃣ OpenAI - Rewrite News (Переписывание через AI)

**Тип:** `@n8n/n8n-nodes-langchain.openAi`

**Функция:** Переписывает новости с помощью GPT-4

**Настройки:**
- Модель: `gpt-4o-mini` (быстрый и дешёвый)
- Temperature: `0.7` (креативность)
- Max Tokens: `500` (длина ответа)

**System Prompt:**
```
Ты - опытный копирайтер для Telegram каналов. 
Твоя задача переписывать новости в интересном и вовлекающем формате.
```

**Альтернативные модели:**
- `gpt-4o` - более качественный, но дороже
- `gpt-3.5-turbo` - самый дешёвый вариант

---

### 6️⃣ Format for Telegram (Форматирование для Telegram)

**Тип:** `n8n-nodes-base.code`

**Функция:** Форматирует ответ AI для Telegram

**Что делает:**
1. Извлекает переписанный текст
2. Добавляет ссылку на источник
3. Добавляет хештеги
4. Настраивает Markdown форматирование

**Пример результата:**

```markdown
Ключевые особенности:
• Улучшенное понимание контекста
• Быстрее в 3 раза
• Поддержка 100+ языков

🔗 [Читать полностью](https://openai.com/blog/gpt5)

#новости #ai
```

## ⚙️ Настройка Credentials

### OpenAI API Key

1. Перейдите в [OpenAI Platform](https://platform.openai.com/api-keys)
2. Создайте новый API ключ
3. В n8n: Settings → Credentials → Add Credential → OpenAI API
4. Вставьте ключ

### Telegram Bot Token

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен
5. В n8n: Settings → Credentials → Add Credential → Telegram API
6. Вставьте токен

### Получение Chat ID канала

1. Добавьте бота в канал как администратора
2. Отправьте сообщение в канал от имени бота
3. Выполните запрос:
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
   ```
4. Найдите `chat":{"id":-1001234567890` в ответе
5. Используйте этот ID в workflow

## 🐛 Troubleshooting

### Ошибка: "Chat not found"

**Решение:**
- Убедитесь, что бот добавлен в канал
- Проверьте правильность Chat ID
- Убедитесь, что бот имеет права администратора

### Ошибка: "OpenAI API rate limit"

**Решение:**
- Увеличьте интервал между запусками
- Используйте более дешёвую модель
- Добавьте обработку ошибок с повторной попыткой

### Ошибка: "RSS feed not accessible"

**Решение:**
- Проверьте доступность RSS URL
- Убедитесь, что n8n имеет доступ к интернету
- Попробуйте другой RSS источник
