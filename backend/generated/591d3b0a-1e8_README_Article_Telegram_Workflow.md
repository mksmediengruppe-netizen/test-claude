## 🚀 Быстрый старт

### Шаг 1: Импорт workflow в n8n

1. Откройте n8n: [http://2.56.240.170:5678](http://2.56.240.170:5678)
2. Создайте новый workflow
3. Нажмите "Import from File"
4. Выберите файл:
   - Для продвинутой версии: `/root/article_to_telegram_workflow.json`
   - Для простой версии: `/root/simple_article_to_telegram.json`

### Шаг 2: Настройка Credentials

#### OpenAI API:
1. В n8n → Credentials → New
2. Выберите "OpenAI API"
3. Введите ваш API ключ от OpenAI

#### Telegram Bot:
1. Создайте бота через @BotFather
2. Получите токен
3. В n8n создайте credentials типа "Telegram API"
4. Введите токен бота

### Шаг 3: Настройка Telegram группы

1. Добавьте бота в группу
2. Сделайте бота администратором
3. Получите chat_id группы:
   - Добавьте @RawDataBot в группу
   - Отправьте `/id`
   - Скопируйте ID (начинается с -100)

4. В workflow замените `-1001234567890` на ваш chat_id

### Шаг 4: Тестирование

1. Активируйте workflow
2. Используйте Manual Trigger
3. Введите URL статьи для теста
4. Проверьте публикацию в Telegram

## 🔧 Конфигурация workflow

### Основные узлы:

1. **Manual Trigger** - Ручной запуск с указанием URL
2. **Schedule Trigger** - Автоматическая публикация по расписанию (10:00, 16:00, 20:00)
3. **HTTP Request** - Получение статьи с сайта
4. **HTML Parser** - Извлечение заголовка и контента
5. **OpenAI Rewriter** - Переписывание через AI
6. **Content Formatter** - Форматирование для Telegram
7. **Telegram Send** - Публикация в группу

### Настройка источников:

**Вариант 1: Ручной ввод URL**
- Используйте Manual Trigger
- Укажите URL статьи

**Вариант 2: Автоматический из Hacker News**
- Используйте Schedule Trigger
- Будет брать случайную статью с первой страницы

**Вариант 3: Кастомный источник**
- Измените URL в узле "HTTP Request"

## 🚨 Устранение неполадок

### Проблема: OpenAI не отвечает
**Решение:**
- Проверьте API ключ в credentials
- Проверьте баланс аккаунта OpenAI
- Увеличьте timeout в настройках узла

### Проблема: Не парсится контент
**Решение:**
- Измените селекторы в HTML Parser
- Добавьте задержку между запросами
- Проверьте, не блокирует ли сайт парсинг

### Проблема: Telegram не отправляет
**Решение:**
- Проверьте chat_id группы
- Убедитесь, что бот - администратор
- Проверьте токен бота
