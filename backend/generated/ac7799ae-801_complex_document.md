## Основные концепции

### Пример кода

Вот простой пример Python-кода, который демонстрирует базовые операции:

```python
# Простой калькулятор на Python

def calculate(a, b, operation):
    """
    Выполняет математическую операцию над двумя числами.
    
    Args:
        a: Первое число
        b: Второе число
        operation: Операция (+, -, *, /)
    
    Returns:
        Результат вычисления
    """
    operations = {
        '+': lambda x, y: x + y,
        '-': lambda x, y: x - y,
        '*': lambda x, y: x * y,
        '/': lambda x, y: x / y if y != 0 else "Ошибка: деление на ноль"
    }
    
    return operations.get(operation, "Неизвестная операция")(a, b)

# Пример использования
result = calculate(10, 5, '+')
print(f"Результат: {result}")
```

## Структуры данных в Python

### Списки (Lists)

Списки — это **упорядоченные** *изменяемые* коллекции объектов:

```python
fruits = ["яблоко", "банан", "апельсин"]
fruits.append("груша")
print(fruits[0])  # Вывод: яблоко
```

### Словари (Dictionaries)

Словари хранят данные в формате **ключ-значение**:

```python
person = {
    "имя": "Алексей",
    "возраст": 25,
    "город": "Москва"
}
print(person["имя"])  # Вывод: Алексей
```
