### 3.2 Cross-Chat Learning

Система учится на **ВСЕХ чатах всех пользователей**:

```python
class CrossChatLearning:
    async def before_task(self, task: str):
        # 1. Найти похожие задачи из ВСЕХ чатов
        similar_tasks = await self.memory.search(
            query=task,
            scope="all_users",  # Не только текущий пользователь!
            include_errors=True,
            include_successes=True,
            limit=10
        )

        # 2. Извлечь уроки
        lessons = []
        for similar in similar_tasks:
            if similar.had_error:
                lessons.append(f"❌ Не делать: {similar.error_description}")
            if similar.was_successful:
                lessons.append(f"✅ Лучшее решение: {similar.solution_pattern}")

        # 3. Добавить в контекст агента
        return self.inject_lessons(lessons)
```

**Результат:** Агент не повторяет ошибки, которые уже были в других чатах.

### 5.2 Алгоритм оценки "Программист vs Агент"

```python
class TaskEstimator:
    # Средние оценки времени по типам задач
    HUMAN_ESTIMATES = {
        "landing_page": {"hours": 3, "rate": 25},      # $75
        "api_endpoint": {"hours": 2, "rate": 25},      # $50
        "bug_fix_simple": {"hours": 0.5, "rate": 25},  # $12.5
        "bug_fix_complex": {"hours": 4, "rate": 25},   # $100
        "parser_script": {"hours": 1.5, "rate": 25},   # $37.5
        "database_migration": {"hours": 2, "rate": 25}, # $50
        "ui_component": {"hours": 1, "rate": 25},      # $25
        "full_feature": {"hours": 8, "rate": 25},      # $200
    }

    async def estimate(self, task: str, agent_cost: float, agent_time: float):
        # Классифицировать задачу
        task_type = await self.classifier.classify(task)

        human = self.HUMAN_ESTIMATES.get(task_type, {"hours": 2, "rate": 25})
        human_cost = human["hours"] * human["rate"]

        return {
            "human_time": f"{human['hours']} часов",
            "human_cost": f"${human_cost}",
            "agent_time": f"{agent_time} мин",
            "agent_cost": f"${agent_cost}",
            "savings_percent": round((1 - agent_cost/human_cost) * 100),
            "speedup": round(human["hours"] * 60 / agent_time)
        }
```

### 12.3 Workflow дизайна

```
ЗАПРОС → RESEARCH → BRIEF → ГЕНЕРАЦИЯ → REVIEW → DELIVERY

1. RESEARCH
   ├── Поиск референсов по запросу
   ├── Анализ 5-10 лучших примеров
   └── Выбор стиля

2. DESIGN BRIEF
   ├── Цветовая палитра (auto-generate)
   ├── Типографика (Google Fonts)
   ├── Структура страницы
   └── Ключевые секции

3. ГЕНЕРАЦИЯ
   ├── HTML + Tailwind CSS
   ├── AI-изображения для hero/features
   ├── SVG иконки
   └── Анимации

4. REVIEW
   ├── Скриншот страницы
   ├── Vision-анализ (Gemini)
   ├── Автоисправление проблем
   └── Mobile адаптация

5. DELIVERY
   ├── Готовые файлы
   ├── Превью ссылка
   └── Исходники
```
