## Код на Python

```python
def fibonacci(n):
    """
    Вычисляет n-е число Фибоначчи.
    
    Args:
        n (int): Порядковый номер числа
        
    Returns:
        int: Число Фибоначчи
    """
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Пример использования
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
